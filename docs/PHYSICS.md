# FINANCIAL THERMODYNAMICS & SYSTEMIC PHYSICS

Project Kessler discards linear risk models (VaR, Black-Scholes) in favor of non-linear phase transition mechanics. Designed for sophisticated quantitative hedge funds, the market is modeled as a high-stress metallurgy problem to rigorously stress-test multi-asset portfolios against true systemic contagion.

## Core Dynamics
1. **Hawkes Self-Exciting Intensity:** Default probabilities are not static. We use a modified Hawkes process where a localized margin call injects an exponential decay spike into the default intensity ($\lambda$) of topologically adjacent nodes.
2. **Gai-Kapadia Contagion Loop:** Contagion propagates through two simultaneous vectors:
   - *Direct:* Counterparty credit failure.
   - *Indirect:* Mark-to-market portfolio devaluation via fire-sale price impact.
3. **Rough Volatility:** Volatility ($\sigma$) does not revert smoothly. We model it as a fractional Brownian motion with long memory (Hurst parameter < 0.5). Volatility clusters and spikes, mirroring the jagged, fractal nature of real panic.
4. **Kyle’s Dynamic Lambda:** Market depth is an illusion. Price impact ($\lambda$) scales dynamically with Order Flow Toxicity (VPIN). As order flow becomes one-sided, liquidity providers withdraw, inducing vacuum crashes (Silence Waves).

## The Strangelet Hypothesis
Just as a single strangelet can catalyze a chain reaction that converts a neutron star into strange matter, a single highly-leveraged default in Kessler's graph can force the entire topological network into an insolvency phase transition. We do not predict the spark; we map the exact flammability of the forest.
